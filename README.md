# Ledger Expense Tracker

A modern minimal expense tracking web app built with React, Vite, and Node.js.

## Features

- Add and manage expenses
- Categorize spending
- Dashboard analytics
- Budget tracking
- Clean minimalist UI

## Tech Stack

Frontend

- React
- Vite
- TailwindCSS

Backend

- Node.js
- Express

Database

- MongoDB (optional)

## Run the project

Install dependencies

npm install

Start development server

npm run dev
